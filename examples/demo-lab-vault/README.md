# Demo lab vault — the framework, lived-in

> This is a small, **realistic** example of a lab's shared brain after a few weeks of Lab Brain use —
> so you can see the [framework](../../framework/README.md) conventions *in use*, not just as blank
> templates. The lab (fictional) studies the host inflammatory response to H9N2 influenza; its
> knowledge is grounded in a real open-access paper
> ([Gui et al. 2017, *Virology Journal*](https://doi.org/10.1186/s12985-017-0683-y), CC-BY).
>
> A mentee ("Alex Rivera") is being onboarded onto the flu-cytokine sub-track; a mentor ("Sam Ortiz")
> runs the check-in loop; the PI ("Maya Rao") reads the rollup.
