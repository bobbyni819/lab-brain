# 01 · The project

**One line:** understand how the host inflammatory response to H9N2 influenza is dysregulated, using
figure-derived cytokine dynamics from the literature to ground a model.

## The question
Which cytokines drive H9N2 lung pathology, and *when* — and does immunomodulation (e.g. calcitriol)
help or hurt? The magnitudes and timing live in figure panels, so reading them reliably is upstream
of everything.
